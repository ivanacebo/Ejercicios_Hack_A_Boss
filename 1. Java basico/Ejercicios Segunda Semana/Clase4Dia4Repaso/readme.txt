El desarrollo del sistema consiste en la implementacion de una clase verdura. Consta de sus atributos junto
con sus constructores basicos ademas de sus getters y setters.

En el desarrollo de su funcionalidad instanciamos primeramente la clase verdura para poder usarla con vectores,
rellenando este con una serie de verduras conocidas.

Las funcionalidades son:

	1. Mostrar por pantalla el nombre y calorias de las distintas verduras.
	2. Modificar dos de las verduras creadas anteriormente para guardar otras en su lugar.
	3. Mostrar por pantalla los datos de todas las verduras que tenemos.
	4. Mostrar por pantalla los datos de las verduras que tengan el color verde.
	5. La información por pantalla se muestra estructurada en varios pasos:
		-- Nombre y calorias de las verduras.
		-- Datos ordenados con salto de linea de todas las verduras.
		-- Datos ordenados con salto de linea de todas las verduras que sean verdes.

Las pruebas de funcionamiento constaran de:

	1. Primera prueba hacer correr el programa.
	2. Modificar los datos de las distintas verduras en su inicio y hacer correr el programa.
	3. Modificar los datos de las dos verduras posteriormente y hacer correre el programa.
	4. Interpretear si los datos que nos muestran por pantalla son los correctos.

Los resultados obtenidos:

	-- Se considera que los resultados obtenidos son satisfactorios cumpliento esto las indicaciones 
	anteriores y mostrando por pantalla los datos correctamente de nuestras verduras aún incluso con 
	las modificaciones de las prubas de los tester.

Los requisitos del sistema son:	
	
	Tener instalado el JDK version 17 de java.

Instrucciones de ejecución:

	Hacer correr el programa en este caso sera suficiente.
